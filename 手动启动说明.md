# StoryForge 手动启动说明

适用于 Windows PowerShell。请把命令里的项目目录换成你自己的下载位置。

如果是第一次运行，先准备依赖：

```powershell
cd C:\你的项目目录\StoryForge
py -m venv .venv
.\.venv\Scripts\pip.exe install -r backend\requirements.txt
cd frontend
npm install
```

之后每次启动，请打开两个终端窗口。

## 1. 启动后端

在第一个终端运行：

```powershell
cd C:\你的项目目录\StoryForge
.\.venv\Scripts\python.exe -m uvicorn backend.app.main:app --host 0.0.0.0 --port 8000
```

看到类似下面内容表示成功：

```text
Uvicorn running on http://0.0.0.0:8000
```

后端本机地址：

```text
http://127.0.0.1:8000
```

后端健康检查：

```text
http://127.0.0.1:8000/health
```

## 2. 启动前端

在第二个终端运行：

```powershell
cd C:\你的项目目录\StoryForge\frontend
npm run dev -- --host 0.0.0.0 --port 5173
```

前端本机地址：

```text
http://127.0.0.1:5173/
```

局域网访问地址格式：

```text
http://你的电脑IP:5173/
```

查看电脑 IP：

```powershell
ipconfig
```

找到 `IPv4 Address`，例如：

```text
http://10.180.22.199:5173/
```

## 3. 关闭服务

在前端和后端对应的终端里按：

```text
Ctrl + C
```

## 4. 注意

如果要让 AI 正常生成内容，后端需要能访问 DeepSeek，并且项目根目录 `.env` 里要配置 `LLM_API_KEY`。
